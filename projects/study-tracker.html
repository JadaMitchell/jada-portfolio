Overview

This web app allows users to create tasks with course, due date, and priority, view progress metrics in a dashboard, and store tasks in a database.

Features

  • Create / update / delete tasks (CRUD)
  • Priority + due date sorting
  • Dashboard counts (total, high priority, due soon, completed)
  • Persistent storage with database
